AES (Advanced Encryption Standard) - Basics

What is AES?

AES is a symmetric encryption algorithm that scrambles your data into an unreadable format. Only someone with the correct secret key can unscramble it back to the original message.

3. Block-Based Encryption (16 bytes = 1 block)

Understanding AES Blocks

AES works on blocks of 16 bytes. Every message is divided into 16-byte chunks, and each chunk is encrypted separately.

4. XOR Operation (Core of AES)

Understanding XOR (Exclusive OR)

AES uses XOR to mix the message with the encryption key. XOR has a special property: if you XOR twice with the same key, you get back the original!

Message: A Key: K Encrypted: A XOR K = C Decrypt: C XOR K = (A XOR K) XOR K = A ‚úì

5. AES Process Steps

How AES Encrypts (Simplified)

  1. Divide message into 16-byte blocks
  2. Generate round keys from the master key
  3. For 10 rounds:
    • SubBytes: Replace each byte using a lookup table
    • ShiftRows: Rearrange the bytes
    • MixColumns: Mix the data (skip in final round)
    • AddRoundKey: XOR with the round key
  4. Output encrypted block (same size as input)
