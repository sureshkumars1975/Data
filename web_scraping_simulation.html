What is Web Scraping?

Web scraping is the automated extraction of data from web pages. Scrapers can gather publicly available information quickly, but scraping can also violate terms of service or privacy expectations.

  • Scraper: automated tool acting like a browser to fetch and parse web pages.
  • Website: publishes HTML that contains the data.
  • User: may unknowingly have their data collected.
