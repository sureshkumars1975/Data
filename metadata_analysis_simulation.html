What is Metadata Analysis?

Metadata is data about data—like timestamps, location tags, or software used. Analyzing metadata can expose patterns or identities even when the main content seems anonymous.

  • File: contains hidden metadata (e.g., camera model, edit history).
  • Analyzer: extracts and inspects metadata.
  • Inference: metadata is used to infer identity, location, or behavior.
