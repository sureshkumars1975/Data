What is an Inference Attack?

An inference attack uses seemingly harmless queries to a database to infer sensitive details about individuals. By combining results and making clever queries, an attacker can deduce hidden information.

  • Database: contains data that is meant to be protected (e.g., health records) and provides query answers.
  • Attacker: submits queries and observes responses to infer sensitive values.
  • Goal: deduce confidential information without directly accessing it.
