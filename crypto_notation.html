Cryptography Notation Reference

A concise table of symbols, expressions and meanings used in cryptography.

Notation
Symbol Expression Meaning
D, K D(K, Y) Symmetric decryption of ciphertext Y using secret key K
D, PRa D(PRa, Y) Asymmetric decryption of ciphertext Y using A's private key PRa
D, PUa D(PUa, Y) Asymmetric decryption of ciphertext Y using A's public key PUa
E, K E(K, X) Symmetric encryption of plaintext X using secret key K
E, PRa E(PRa, X) Asymmetric encryption of plaintext X using A's private key PRa
E, PUa E(PUa, X) Asymmetric encryption of plaintext X using A's public key PUa
K — Secret key
PRa — Private key of user A
PUa — Public key of user A
MAC, K MAC(K, X) Message authentication code of message X using secret key K
GF(p) — Finite field of order p (prime). Set $\mathbb{Z}_p$ with arithmetic modulo $p$.
GF(2n) — Finite field of order $2^n$.
$\mathbb{Z}_n$ — Set of integers $\{0,1,\dots,n-1\}$ (integers modulo $n$)
$\gcd$ gcd(i, j) Greatest common divisor: $\gcd(i,j)$
Congruence $a \equiv b \pmod m$ $a$ is congruent to $b$ modulo $m$ (same remainder)
Not congruent $a \not\equiv b \pmod m$ Not congruent modulo $m$
dlog $\mathrm{dlog}_{a,p}(b)$ Discrete logarithm of $b$ to base $a$ modulo $p$
$\varphi(n)$ — Euler's totient function: count of integers $\lt n$ relatively prime to $n$
$\Sigma$ $\sum_{i=1}^n a_i = a_1 + a_2 + \dots + a_n$ Summation
$\Pi$ $\prod_{i=1}^n a_i = a_1 \cdot a_2 \cdot \dots \cdot a_n$ Product
$i \mid j$ — $i$ divides $j$ (no remainder when $j/i$)
$|a|$ — Absolute value of $a$
· A \cdot (a_1,a_2,\dots,a_k) Integer A corresponds to sequence $(a_1,a_2,\dots,a_k)$ (context-dependent)
} — Concatenation: x } y means x concatenated with y
≈ — Approximately equal
⊕ — Exclusive-OR (XOR). Bitwise XOR for multi-bit variables
Floor $\lfloor x \rfloor$ Largest integer less than or equal to $x$
∈ x ∈ S Element x is contained in set S
