What is Eavesdropping / Interception?

Eavesdropping means an attacker reads the data being sent between two parties without altering it. Interception means the attacker can also change the message before forwarding it.

  • Client: Sends a request to the server.
  • Server: Sends back a response.
  • Attacker: Watches messages transit and can modify them if the connection is not secured.
