What is RMI?

Remote Method Invocation (RMI) is a Java technology that allows an object running in one Java Virtual Machine (JVM) to invoke methods on an object in another JVM. RMI hides the networking details, allowing the programmer to call remote methods as if they were local.

  • Client: Calls a method on a remote interface (via a proxy/stub).
  • Server: Hosts the remote object and executes the invoked method.
  • Stub/Skeleton: Stub (client-side proxy) marshals calls to the server; skeleton (server-side dispatcher) unmarshals them and invokes the object.
