What is RPC?

Remote Procedure Call (RPC) lets a program ask another program (often on a different machine) to execute a function and return the result. The caller (client) sends a request message, the callee (server) executes the requested function, and then sends a response.

  • Client: Makes a request (calls a remote function).
  • Server: Receives the request, executes the function, and returns the result.
  • Request/Response: The data flows through a transport layer (e.g., HTTP, TCP, or gRPC).
